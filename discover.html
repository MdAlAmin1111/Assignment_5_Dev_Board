Question-1: What are the different ways to select an element in the DOM?

There are several ways to select elements in the DOM using JavaScript. One method is document.getElementById("id"), which grabs an element by its unique ID. Another way is using document.getElementsByClassName("class"), which selects all elements that share a specific class, return a collection. If we want to target elements based on their tag name, we can use document.getElementsByTagName("tag"). Additionally, document.querySelector("selector") allows you to select the first element that matches a given CSS selector, while document.querySelectorAll("selector") selects all elements that match.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets or sets the HTML inside an element, including tags. innerText gives the visible text, ignoring hidden text or styles. textContent gets all the text inside an element, even if it's hidden, without caring about styles. So, innerHTML includes tags, innerText shows only visible text, and textContent gets all text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where you attach a single event listener to a parent element instead of adding individual listeners to each child element. The parent listens for events that bubble up from the child elements. This way, we can handle events for many child elements more efficiently, even if they are added dynamically. It reduces the number of event listeners and improves performance.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in the DOM where an event starts from the target element and then "bubbles up" to its parent elements, one by one, until it reaches the root of the document. This allows parent elements to listen for events that happen on their child elements. For example, if a click event happens on a button inside a div, the event will first trigger on the button and then bubble up to the div, body, and other ancestors.

Question-5: How do you create, add, and remove elements using JavaScript?

To create, add, and remove elements using JavaScript, we can use a few simple methods. First, use document.createElement("tagName") to create a new element. Then, to add it to the DOM, use parentElement.appendChild(childElement) to attach it to a parent element. To remove an element, we can call parentElement.removeChild(childElement) on its parent element. These methods help you dynamically manipulate elements on our web page as needed.
